# Cracks and Potholes in Road Images

## Overview

Cracks and Potholes in Road Images is a segmentation package built from multiclass distress masks.

- Task: segmentation
- Images: 2235
- Annotations: 324370
- Classes: 2 (crack, pothole)
- Annotation format: paired PNG masks

## Source Dataset

- Source repository: https://github.com/biankatpas/Cracks-and-Potholes-in-Road-Images-Dataset
- Project page: https://biankatpas.github.io/Cracks-and-Potholes-in-Road-Images-Dataset/
- Mendeley record: https://data.mendeley.com/datasets/t576ydh9v8/4
- DOI: 10.17632/t576ydh9v8.4

## Notes

- The converted source folder provides multiclass semantic masks in `masks/`.
- Label encoding in this package is: 0=background, 1=crack, 2=pothole.
- The `masks_with_roadlanes/` folder is intentionally excluded.
